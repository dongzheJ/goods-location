# 📦 GeoGoods - 仓储货位一键找货系统

这是一个简单好用的**仓库/店内商品找货系统**。它可以把复杂的仓库平面图变成像“电子地图”一样，输入商品名称，它就能立刻告诉你货在哪。

---

## ✨ 它能做什么？

### 1. 🔍 一键导航（找货不迷路）
在顶部的搜索框输入商品的**名字**或者**条码**，地图上对应的货架就会立刻**跳动大红针**并高亮显示。即使是再大的仓库，也能一秒发现目标。

### 2. 🗺️ 电子地图（丝滑缩放）
你可以像用手机地图一样，随意用鼠标滚轮缩放图纸，或者拖拽查看细节。货架位置一目了然。

### 3. 🪄 智能标注（快速建档）
*   **框选标注**：开启“Mapping Mode”后，直接在图纸上按住鼠标左键画个圈，就能定义一个新的货架。
*   **魔法识别**：点击“魔法棒”图标，系统会自动根据图纸上的颜色（红、绿、蓝、紫）自动识别出一排排的货架，省掉手动标记的麻烦。

---

## 🛠️ 文件结构（哪里存着数据？）

对于非技术人员，你只需要关注 `src/data` 文件夹下的这两个文件：

*   📄 **products.json (商品清单)**：这里存着你的商品信息，比如 ID、名字和它在哪个货位（如 `B-28`）。
*   📄 **shelves.json (货架地图)**：这里存着地图上每个货架的坐标。

---

## 🚀 如何启动？

1.  **准备环境**：确保电脑安装了 Node.js。
2.  **安装命令**：在项目文件夹打开终端，运行 `npm install`。
3.  **启动系统**：运行 `npm run dev`，然后在浏览器打开显示的地址（通常是 `http://localhost:5173`）即可使用。
